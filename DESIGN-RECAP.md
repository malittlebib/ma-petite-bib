## 🚀 COMMENT ÇA FONCTIONNE

Chaque page charge maintenant :

```html
<head>
  <!-- Police Inter -->
  <link href="https://fonts.googleapis.com/.../Inter" rel="stylesheet">
  
  <!-- Font Awesome -->
  <link rel="stylesheet" href=".../font-awesome/6.4.0/css/all.min.css">
  
  <!-- 🎨 NOUVEAU : Design System Global -->
  <link rel="stylesheet" href="style-global.css">
  
  <!-- Styles spécifiques de la page -->
  <style>
    /* Styles custom pour cette page uniquement */
  </style>
</head>
```

### Avantages
- ✅ **Cohérence** : Même look partout
- ✅ **Maintenance** : 1 seul fichier à modifier
- ✅ **Performance** : CSS en cache
- ✅ **Flexibilité** : Peut overrider si besoin
