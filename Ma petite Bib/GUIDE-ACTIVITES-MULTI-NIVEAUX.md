# 📅 GUIDE RAPIDE : Créer des activités différenciées

## 🎯 COMMENT FAIRE ?

### **Scénario 1 : Activités différentes au MÊME créneau**

**Exemple** : Lundi 9h30-10h00
- PS fait de la pâte à modeler
- MS fait du graphisme

**MÉTHODE** :

1️⃣ **Créer l'activité PS**
   - Jour : Lundi
   - Heure : 9h30 - 10h00
   - Titre : "Pâte à modeler"
   - Niveaux : Décocher "Tous" → Cocher "🟡 PS"
   - Enregistrer

2️⃣ **Créer l'activité MS** (AU MÊME CRÉNEAU)
   - Jour : Lundi
   - Heure : 9h30 - 10h00
   - Titre : "Graphisme boucles"
   - Niveaux : Décocher "Tous" → Cocher "🔴 MS"
   - Enregistrer

**RÉSULTAT dans l'emploi du temps** :

```
┌─────────────────┐
│ Lundi 9h30      │
├─────────────────┤
│ 🟡 Pâte à       │  ← Cliquez pour voir/éditer activité PS
│    modeler      │
│                 │
│ 🔴 Graphisme    │  ← Cliquez pour voir/éditer activité MS
│    boucles      │
└─────────────────┘
```

⚠️ **IMPORTANT** : Les deux activités s'empilent dans la même case. C'est normal !

---

### **Scénario 2 : Activité commune à PLUSIEURS niveaux**

**Exemple** : PS + MS font des ateliers maths ensemble

**MÉTHODE** :

1️⃣ **Créer l'activité**
   - Titre : "Ateliers mathématiques"
   - Niveaux : Décocher "Tous" → Cocher "🟡 PS" ET "🔴 MS"
   - Enregistrer

**RÉSULTAT** :
```
┌─────────────────┐
│ 🟡🔴 Ateliers   │  ← Les deux emojis montrent PS+MS
│  mathématiques  │
└─────────────────┘
```

## 💡 ASTUCES

### **Dupliquer une activité**
Si vous avez créé "🟡 Pâte à modeler" le lundi et voulez la mettre aussi le jeudi :
1. Survolez l'activité
2. Cliquez sur l'icône "Copie" 📋
3. Cochez "Jeudi"
4. Valider

### **Changer le niveau d'une activité**
1. Cliquez sur l'activité dans la grille
2. Modifiez les niveaux
3. Enregistrer

### **Voir rapidement qui fait quoi**
Les emojis vous donnent l'info en un coup d'œil :
- 🟡 = PS uniquement
- 🔴 = MS uniquement
- 🔵 = GS uniquement
- 🟡🔴 = PS + MS
- 🔴🔵 = MS + GS
- 🌈 = Tous

## ❓ FAQ

**Q : Peut-on avoir 3 activités différentes au même créneau (PS, MS, GS) ?**
R : Oui ! Créez 3 activités au même horaire, une pour chaque niveau.

**Q : Comment voir seulement les activités PS ?**
R : Pour l'instant, regardez les 🟡. Une fonction de filtre arrive bientôt !

**Q : L'emploi du temps devient chargé, comment faire ?**
R : Réduisez la taille de police ou imprimez en paysage.
