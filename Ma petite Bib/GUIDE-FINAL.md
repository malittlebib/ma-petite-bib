## 🚀 Utilisation en 3 étapes

### **Étape 1 : Créer une activité**

Ouvrez le **Cahier Journal** → Cliquez sur **"+"**

### **Étape 2 : Le titre déclenche les suggestions**

Tapez : `Jeu des rimes` → **Attendez 1 seconde**

**Une bulle jaune apparaît** :
```
💡 Suggestions automatiques :
Associer des mots ayant la même rime  [+ Ajouter]
```

Cliquez **"+ Ajouter"** → Tag violet créé !

### **Étape 3 : Recherche manuelle (optionnel)**

Dans **"🔍 Rechercher..."**, tapez `compter`  
→ Liste de compétences  
→ Cliquez **"+ Ajouter"**

---

### **Dans le formulaire** :
```

└─ 🔍 Rechercher...  ← Recherche manuelle
```

### **Sur les activités** :
```
┌─────────────────────────┐
│ Activité              ✅ 2│ ← Badge vert cliquable

```

Clic sur badge → Modal détaillée
