## 💡 QUESTIONS FRÉQUENTES

### **Mes données sont-elles sauvegardées ?**

Oui ! Tout est sauvegardé **dans votre navigateur** (localStorage).

⚠️ **Attention** : Si vous videz le cache du navigateur, vous perdez tout.

**Conseil** : Exportez régulièrement en PDF pour sauvegarder.

---

### **Puis-je utiliser de vraies données ?**

**OUI**, vos données restent **100% privées** :
- Stockées uniquement sur votre appareil
- Jamais envoyées à un serveur
- Aucun compte requis

**Mais conseil** : Utilisez des données fictives pour les tests.

---

### **L'application fonctionnera-t-elle hors ligne ?**

Oui ! L'application est une **PWA** (Progressive Web App).

Après la première visite, elle fonctionne sans Internet.

---

### **Combien de temps dure le test ?**
